# AI Crime Detection & Prediction System

## Overview

This project is an **AI-powered Crime Detection and Prediction Dashboard** that helps analyze crime patterns and visualize high-risk zones using modern web technologies and machine learning.

The system integrates **crime prediction analytics, CCTV surveillance detection, heatmap visualization, and zone-wise crime statistics** to assist authorities in monitoring potential crime hotspots.

---

## Features

### 1. High Risk Zone Identification

Displays the most crime-prone areas in Visakhapatnam based on predicted crime patterns.

### 2. Zone-Wise Crime Prediction

Interactive bar chart showing predicted crime counts for different city zones.

### 3. Crime Heatmap Visualization

Map-based visualization showing crime intensity in different regions.

### 4. CCTV Surveillance Detection

Upload CCTV footage and detect suspicious activities using AI-based video processing.

### 5. Real-Time Dashboard

Interactive dashboard with live clock and alert system for monitoring.

## Installation

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/crime_detection_AIML.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

4. Open browser

```
http://127.0.0.1:5000
```
